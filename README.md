## Requirements

- node
- npm
- nodemon (optional)
- `.env` file with the configuration

## Setup

- clone repository
- run 'npm install'

## Run project locally

```bash
npm run dev
```

## Script used by Heroku to run the app

```bash
npm run start
```

## Project structure

- src/database
- src/routes
- src/index.js
- static
